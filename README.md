# triviaApi-react
project 1
This project is client side rending for the TriviaAPI.
# Setup process in local machine 
- git clone 
- npm install 
- npm start 
# Deploy process
- npm run build 
- upload all files and folders from the build folder to remote such as AWS S3 buckets
